Add a custom shortcut link to the WordPress toolbar with this plugin.

== Description ==

This plugin allows you to add a custom shortcut link to the WordPress toolbar.
It has two simple options:

*   Custom Label
*   Custom URL


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. A new menu item will be added - Custom Links under settings which will take you to plugin's page
4. You can a new custom shortcut menu there.
